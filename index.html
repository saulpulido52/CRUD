Que es un "Crud"

En general, un CRUD es la forma más común de realizar operaciones básicas de gestión de datos en aplicaciones web y móviles. Cada una de las operaciones básicas puede realizarse a través de una interfaz de usuario que permita al usuario interactuar con la base de datos, lo que permite a los usuarios crear, leer, actualizar y eliminar registros de datos en la base de datos
